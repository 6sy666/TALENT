==========================
Overview
==========================

The core components provide the foundation for all TALENT functionality:

- **Utils**: Essential utilities for training, evaluation, and configuration
- **Data**: Data loading, preprocessing, and transformation capabilities  
- **Method Base**: The base class that all model implementations inherit from

These components work together to provide a consistent and robust framework for tabular machine learning.
